FAQ

Common questions

Is DocBrain really free?
DocBrain is open source and free for production use under BSL 1.1. The only restriction: you cannot offer DocBrain as a hosted/managed service to third parties. At 5,000 GitHub stars or by Jan 1, 2028, it converts to Apache 2.0.
Does my data leave my infrastructure?
When self-hosted, your docs and queries never leave your infrastructure. LLM calls (Anthropic, OpenAI, Bedrock) go to their APIs — use Ollama for 100% local, air-gapped deployment with zero data egress.
Which LLMs does DocBrain support?
15+ providers: Anthropic (Claude), OpenAI (GPT-4), AWS Bedrock, Vertex AI, Azure OpenAI, Groq, OpenRouter, Together, DeepSeek, Mistral, xAI, Gemini, Cohere, and Ollama for 100% local inference. Swap via config — no code changes.
Does DocBrain learn from feedback?
Yes. Every thumbs up/down updates procedural retrieval rules and feeds Autopilot gap detection. Optionally, enable the learning pipeline to fine-tune embeddings from feedback — the system learns what "similar" means for your team's docs and vocabulary. Starts with zero extra infra; scale to full fine-tuning when ready.
How long does setup take?
Docker Compose: about 5 minutes. Run bash scripts/setup.sh and you get PostgreSQL, OpenSearch, Redis, migrations, and sample ingest. Add your LLM API key and point at your Confluence/GitHub.
